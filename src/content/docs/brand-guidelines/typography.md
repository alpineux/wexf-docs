## Fonts

This site uses **system fonts** for fast loading and a native feel:

- **Body text:** System UI / San Francisco / Segoe UI
- **Code:** SF Mono / Cascadia Code / Menlo

## Type scale

| Element | Size | Weight |
|---------|------|--------|
| H1 | 2rem | 700 (Bold) |
| H2 | 1.35rem | 600 (Semibold) |
| H3 | 1.1rem | 600 (Semibold) |
| Body | 1rem | 400 (Regular) |
| Small / Captions | 0.875rem | 400 (Regular) |

## Best practices

- Use **one heading level per section** — don't skip from H1 to H3
- Keep paragraphs short — aim for 2-4 sentences
- Use **bold** for emphasis, *italics* sparingly
- Line length should stay under ~75 characters for readability
