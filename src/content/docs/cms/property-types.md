# Property Types

## What This Is

This is a collection that stores different categories or types of properties. Think of it like a filing system that helps organize properties into groups such as "Residential," "Commercial," "Industrial," or other property classifications.

## What It Does

This collection creates a standardized list of property types that can be used throughout your website. When you add a new property listing, you can select from these predefined types to keep everything organized and consistent. This makes it easier for visitors to filter and find the specific kind of property they're looking for.

## Key Details

- **Collection Name**: Property Types
- **URL Pattern**: `/property-types/[item-name]`
- **Current Items**: 5 property types are set up
- **Fields**: No custom fields are currently configured for this collection

## Things to Know

This collection appears to be set up but doesn't have any custom fields defined yet. Most property type collections would typically include fields like:
- A name or title for the property type
- A description explaining what properties fall into this category
- Perhaps an icon or image to represent the type visually

With 5 items already created, the basic structure is in place and ready to be connected to your main property listings. This helps keep your property categories organized and makes it easier for visitors to browse by the type of property they're interested in.
